# Botium Toys Security Audit

## Objective

Conduct a security audit to identify risks, missing controls, and compliance gaps.

## Risk Rating

8/10 (High Risk)

## Key Findings

- No encryption
- No backups
- No IDS
- Weak password controls
- No disaster recovery plan

## Frameworks Used

- NIST CSF
- PCI DSS
- GDPR
- SOC

# Botium Toys Security Audit

## Objective

Conduct a cybersecurity audit to identify risks, missing controls, and compliance gaps.

## Risk Rating

8/10 (High Risk)

## Findings

### Missing Controls

- Least Privilege
- Separation of Duties
- Encryption
- Intrusion Detection System (IDS)
- Backups
- Disaster Recovery Plan
- Password Management System

### Existing Controls

- Firewall
- Antivirus
- CCTV Monitoring
- Physical Locks
- Fire Detection Systems

## Compliance Frameworks

- PCI DSS
- GDPR
- SOC
- NIST Cybersecurity Framework

## Recommendations

1. Implement encryption for cardholder data
2. Deploy an IDS solution
3. Create disaster recovery procedures
4. Implement backup strategies
5. Apply least privilege access controls
6. Improve password policies

## Conclusion

Botium Toys presents a high security risk due to missing access controls, lack of encryption, and absence of disaster recovery capabilities.
